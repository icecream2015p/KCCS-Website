# KCCS-Online
## 概要
計算カードカウントシステムKCCSのブラウザ版です。計算カードのタイム計測を目的としたWebサービスです。
様々な計算カード専用の機能を提供するはずなので、もし良かったら使ってみてください。
## プログラミング言語
TurboWarp(Scratch)
